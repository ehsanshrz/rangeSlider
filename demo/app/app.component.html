RangeSlider base on NpnSlider

A Multi Range Slider Component
Github

NpnSlider is a multi range slider component using Angular v6.0. It can be used either as a multi range or a single range slider. By default, it is a multi range slider.

Basic Slider (Multi Range)

{{value}}

Selected Values: {{currentValues | json}}

{{codeSnippetText}}
```html <npn-slider [min]="2006" [max]="2020" (onChange)="onSliderChange($event)"></npn-slider> ```

Single Range Slider

{{codeSnippetText}}
```html <npn-slider [multiRange]="false" [min]="2006" [max]="2020" [minSelected]="2010"></npn-slider> ```

By using multiRange property, you can convert the slider as a single range slider

Multi Range Slider - with preselected values

{{codeSnippetText}}
```html <npn-slider [min]="1000" [max]="5000" [minSelected]="2000" [maxSelected]="4000" [step]="500" [showStepIndicator]="true"> </npn-slider> ```

Disabled Slider

{{codeSnippetText}}
```html <npn-slider [min]="1000" [max]="5000" disabled></npn-slider> ```

Slider With No Tooltip and Values

{{codeSnippetText}}
```html <npn-slider hide-tooltip hide-values [min]="1000" [max]="2000" [minSelected]="1200" [multiRange]="false"></npn-slider> ```
